## Chapter02

-- body 내부 태그 --
<h1> ~ <h6>: 제목 타이틀을 만들때 쓰임, 글씨가 커지고 두꺼워짐 1이 가장 크고 6으로 갈 수록 글씨가 작아짐.
<div>: 영역을 나누는데 쓰임
<p>: 문단을 나누는데 쓰임(div 태그와 거의 비슷, 위아래 공간이 생김)
<img>: 이미지 넣는데 사용, 단일 태크
<input>: 사용자에게 입력을 받는 태그, 단일 태그
<button>: 버튼만드는 태크
<a>: 링크를 연결 -> 여러 페이지로 이동가능

-- head 내부 태그 --
<meta> : 웹서버와 웹브라우저 사이에 주고받는 정보를 정의하는데 사용됨 문서 자체의 특성을 나타냄
<title>: 웹사이트가 열리면 상단에 생기는 tab에 이름을 나타냄
<link> : 필요한 스타일이나 아이콘등 외부 문서를 html문서에 연결시키는데 사용됨

## Chapter03

-- HTML 태그에 스타일을 주는 법
1. HTML 태그에 직접 주는방법
2. id를 이용한 방법 (#어쩌고)
3. class를 이용한 방법 (.어쩌고)
4. 복합선택자
    1) 일치 선택자 -> 두가지 조건을 동시에 만족하는 요소 선택
        ex) div.new-style{}
    2) 자식 선택자 -> div의 자식요소중 class="new-style" 선택
        ex) div>.new-style{}
    3) 후손 선택자 -> div 후손요소들 중에서 class="new-style" 선택
        ex) div .new-style{}

-- 자주 사용되는 CSS 스타일
color : 텍스트의 색을 바꿈
background-color: 배경색을 바꿈
border : 테두리를 그려줌, 3개의 인자를 받음 (두께, 스타일, 색깔)
width / height: 넓이와 높이를 지정하는데 쓰임
text-align : 정렬 (left, center, right)
margin : 내 영역 밖에 공간을 줌 (top, left, right, bottom)
padding: 내 영역 안에서 공간을 줌 (top, left, right, bottom)


## Chapter04

-- CDN 이란?
CDN(Content Delivery Network)은 지리적으로 분산된 여러 개의 서버입니다.
웹 콘텐츠를 사용자와 가까운 곳에서 전송함으로써 속도를 높이는 방식입니다.
전 세계 데이터센터는 파일 복사본을 임시로 저장하는 프로세스인 캐싱을 사용.
따라서 사용자는 가까운 서버를 통해 웹 활성화 디바이스 또는 브라우저에서 인터넷 콘텐츠에 빠르게 접속가능.
CDN은 웹 페이지, 이미지, 비디오 등의 콘텐츠를 사용자의 물리적인 위치와 가까운 프록시 서버에서 캐싱


## Chapter05

-- static
웹사이트의 기본 속성값, top right bottom left 등 위치 속성은 무시.

-- relative
HTML 태그가 있는 위치에서 left right top bottom 값을 통해 움직임(있는자리 기준에서 위치를 옮김)

-- absolute
부모영역에서 left right top bottom 을 이용해 주어진 위치로 움직임(부모 절대값 기준으로 옮김)

* 단! 여기서 부모란?: 부모 태그가 relative, absolute, fixed 속성 중 하나여야 함.
 만약 부모태그가 해당 속성을 가지고있지 않다면 body태그 기준으로 움직임 왜냐하면 body태그는 relative를 기본 속성으로 가지고 있기 때문이다

-- fix
absolute 와 비슷하지만 스크롤로 내려도 그 위치에 고정, 무조건 브라우저 창 기준

-- sticky
relative 와 비슷하지만 스크롤로 내리면 fixed 처럼 위치에 고정

-- flex box 특징
1. display: flex 는 모든 요소를 가로로 둔다.
2. display: flex 는 부모한테 적용을 하고 자손을 컨트롤 한다.
3. justify-content: 가로로 요소들을 움직인다.
4. align-items: 세로로 요소들이 움직인다.
5. flex-direction:column : 가로로 정렬된 요소를 세로로 바꾸고 justify-content 는 세로로 align-items 는 가로로 방향으로 바뀐다.

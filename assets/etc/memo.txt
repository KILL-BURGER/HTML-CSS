## Chapter02

-- body 내부 태그 --
<h1> ~ <h6>: 제목 타이틀을 만들때 쓰임, 글씨가 커지고 두꺼워짐 1이 가장 크고 6으로 갈 수록 글씨가 작아짐.
<div>: 영역을 나누는데 쓰임
<p>: 문단을 나누는데 쓰임(div 태그와 거의 비슷, 위아래 공간이 생김)
<img>: 이미지 넣는데 사용, 단일 태크
<input>: 사용자에게 입력을 받는 태그, 단일 태그
<button>: 버튼만드는 태크
<a>: 링크를 연결 -> 여러 페이지로 이동가능

-- head 내부 태그 --
<meta> : 웹서버와 웹브라우저 사이에 주고받는 정보를 정의하는데 사용됨 문서 자체의 특성을 나타냄
<title>: 웹사이트가 열리면 상단에 생기는 tab에 이름을 나타냄
<link> : 필요한 스타일이나 아이콘등 외부 문서를 html문서에 연결시키는데 사용됨

## Chapter03

-- HTML 태그에 스타일을 주는 법
1. HTML 태그에 직접 주는방법
2. id를 이용한 방법 (#어쩌고)
3. class를 이용한 방법 (.어쩌고)
4. 복합선택자
    1) 일치 선택자 -> 두가지 조건을 동시에 만족하는 요소 선택
        ex) div.new-style{}
    2) 자식 선택자 -> div의 자식요소중 class="new-style" 선택
        ex) div>.new-style{}
    3) 후손 선택자 -> div 후손요소들 중에서 class="new-style" 선택
        ex) div .new-style{}

-- 자주 사용되는 CSS 스타일
color : 텍스트의 색을 바꿈
background-color: 배경색을 바꿈
border : 테두리를 그려줌, 3개의 인자를 받음 (두께, 스타일, 색깔)
width / height: 넓이와 높이를 지정하는데 쓰임
text-align : 정렬 (left, center, right)
margin : 내 영역 밖에 공간을 줌 (top, left, right, bottom)
padding: 내 영역 안에서 공간을 줌 (top, left, right, bottom)

-- 입력모드
1. i, a, o

-- 노말모드
1. esc 를 누르면 된다.
2. 방향키로 커서를 움직일 수 있다.
3. h, j, k, l 로도 움직일 수 있다.
4. w: 다음 단어 시작.
5. e: 다음 단어 끝.
6. ctrl + f : 페이지 다운
7. ctrl + b : 페이지
8. x : 글자 삭제
9. dd : 행 삭제
10. u : undo
11. ctrl + r : redo
12. yy : 행복사
13. p : 붙여넣

-- 비주얼 모드
1. v 키를 누르면 된다.
2. d: cut
3. y: copy
4. p: paste 붙여넣기
5. "+p: 외부 코드 복붙을 할 경우

-- 명령 모드
:w -> 파일 저장
:q -> 파일 저장없이 나가기
